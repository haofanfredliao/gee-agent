Reference diagnostic Python snippet (AOI unresolved branch):
```python
gaul = ee.FeatureCollection("FAO/GAUL/2015/level2")
cn = gaul.filter(ee.Filter.eq("ADM0_NAME", "China"))
gd = cn.filter(ee.Filter.Or(
    ee.Filter.eq("ADM1_NAME", "Guangdong"),
    ee.Filter.eq("ADM1_NAME", "Guangdong Sheng")
))

adm2_names = ee.List(gd.aggregate_array("ADM2_NAME")).distinct().sort()
name_sample = adm2_names.slice(0, 20)
alias_hit = adm2_names.contains("Nanshan") \
    .Or(adm2_names.contains("Nanshan Qu")) \
    .Or(adm2_names.contains("南山区"))

print("DIAG_CANDIDATE_COUNT", adm2_names.size())
print("DIAG_ADM2_SAMPLE", name_sample)
print("DIAG_ALIAS_HIT", alias_hit)
print("WARN_APPROX_AOI", "Nanshan not found in FAO/GAUL/2015/level2 after China/Guangdong filter")
```

Reference JavaScript snippet (GEE Code Editor):
```javascript
var aoi = ee.Geometry.Rectangle([113.95, 22.28, 114.05, 22.38]); // replace if available
var startDate = '2022-07-01';
var endDate = '2022-09-30';

var dw = ee.ImageCollection('GOOGLE/DYNAMICWORLD/V1')
  .filterBounds(aoi)
  .filterDate(startDate, endDate);

var label = dw.select('label').mode();
var classArea = ee.Image.pixelArea().addBands(label).reduceRegion({
  reducer: ee.Reducer.sum().group({
    groupField: 1,
    groupName: 'class'
  }),
  geometry: aoi,
  scale: 10,
  maxPixels: 1e13
});

print('class_area_grouped', classArea);
```

- Returning user guidance as "recommended prompt/template" instead of asking for concrete missing inputs.
- Continuing with guessed AOI after place resolution failed.
- Planning two execute steps where step-2 only re-computes/re-sorts already available results.
- Using ee.List.sort('value', False), ee.List.sort(lambda..., False), or ee.List.sortBy(...) in Python.
- Continuing to step-3 computation after step-2 AOI/boundary failure.
- Hiding blocker status in verbose text without explicit blocker token.

- Changing required diagnostic print keys to free-form prose, which breaks blocker detection.
